## 🚀 Local Setup

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Generate datasets & train models (run once)
python setup.py

# 3. Launch the app
streamlit run "🏠_Home.py"
```

Open http://localhost:8501 in your browser.

## ☁️ Deploy to Streamlit Community Cloud (Recommended — Free)

### Step 1 — Push to GitHub

```bash
git init
git add .
git commit -m "Initial commit: ML & NN Streamlit project"
git branch -M main
git remote add origin https://github.com/<your-username>/<repo-name>.git
git push -u origin main
```

### Step 2 — Deploy on Streamlit Cloud

1. Go to **https://share.streamlit.io** and sign in with GitHub
2. Click **"New app"**
3. Select your repository and branch (`main`)
4. Set **Main file path** to: `🏠_Home.py`
5. Click **"Deploy!"**

> Streamlit Cloud automatically installs `requirements.txt` and caches model files.
> Add a call to `setup.py` inside `🏠_Home.py` using `st.cache_resource` to train
> models on first cold start.

## 🌐 Alternative Deployment Options

### Docker

```dockerfile
FROM python:3.10-slim
WORKDIR /app
COPY . .
RUN pip install -r requirements.txt && python setup.py
EXPOSE 8501
CMD ["streamlit", "run", "🏠_Home.py", "--server.port=8501", "--server.address=0.0.0.0"]
```

```bash
docker build -t ml-streamlit .
docker run -p 8501:8501 ml-streamlit
```

### Railway / Render

- Connect your GitHub repository
- Start command: `python setup.py && streamlit run "🏠_Home.py" --server.port=$PORT --server.address=0.0.0.0`

### Google Cloud Run / AWS App Runner

Use the Docker image above and push to ECR / Artifact Registry.
